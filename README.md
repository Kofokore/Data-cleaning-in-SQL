# World Layoff Data Cleaning
Several Companies did a mass layoff of workers with the number of workers laid off, industry, country, career stage, percentage laid off all documented. This project is carried out using SQL to clean, transform and prepare the layoffs data set for exploratory analysis. 
## Data Cleaning Structure
1. Removing duplicates 
2. Standardizing the data (trimming and replacing values)
3. Removing null or blank values
4. Removing un-wanted columns
## Tools Used
1. My SQL Workbench
2. SQL (Windows function, CTEs, Join function)
